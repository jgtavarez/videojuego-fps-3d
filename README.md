## 🚀 Instalación y Ejecución

### Requisitos
- Node.js 16+ 
- npm o yarn

### Desarrollo Local
```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# El juego estará disponible en http://localhost:3000
```

### Construcción para Producción
```bash
# Construir proyecto
npm run build

# Vista previa de la construcción
npm run preview
```
